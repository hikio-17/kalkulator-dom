Cara Penggunaan

Pada Kalkulator menerima 2 inputan yaitu value 1 sebagai (a) dan value 2 sebagai (b)

  1. Masukkan value kemudian klik tombol add => (a)
  2. Pilih Operator (+, -, x, /)
  3. Masukkan value kemudian klik tombol add => (b)
  4. Tombol add akan disable ketika (a) dan (b) telah diinputkan
  5. Klik tombol Submit untuk mendapatkan hasil perhitungan
  6. Klik tombol Reset untuk melakukan perhitungan lagi

Validasi

  • Ketika inputan kosong dan user klik tombol add akan mucul alert dengan pesan "inputan tidak boleh kosong"
  • Ketika user menekan operator perhitungan namun belum memasukkan inputan (a) akan muncul alert dengan pesan "Inputkan angka terlebih dahulu!"
  • Ketika user menekan tombol Submit namun belum memasukkan salah satu inputan (a) atau (b) akan mucul alert dengan pesan "Masukkan inputan angka terlebih dahulu!"
